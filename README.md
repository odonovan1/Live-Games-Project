### ***Description***
#### This project will be built using react.js and postgres and javascript. It will allow users to create accounts and have access to live odds and scores using odds API. Users will be able to save matches or games and talk with others on about said game through a review style interface present on the users navbar. The user in the future may be able to place mock bets to see how there picks would play out.
